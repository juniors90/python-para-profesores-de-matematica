Python contruye las soluciones características sigueiendo las reglas de PEMDAS/COPDMAS:

  • PEMDAS

    • paréntesis,

    • exponentes,

    • multiplicaión,

    • división,

    • adición,

    • sustracción.

o como tambien suelen llamar algunod profesores

  • COPDMAS:

    • corchetes,

    • paréntesis,

    • división y multiplicación,

    • adición y sustración.
